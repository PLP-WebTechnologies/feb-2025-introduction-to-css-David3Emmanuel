What is CSS?

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Key CSS Concepts

  • Selectors - target HTML elements to style
  • Properties - what aspects you want to change
  • Values - how you want to change them
